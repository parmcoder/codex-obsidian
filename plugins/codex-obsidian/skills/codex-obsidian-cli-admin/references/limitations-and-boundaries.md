# Limitations and Boundaries

## This skill supports

- official desktop Obsidian runtime-admin command families
- read-only runtime inventory for plugins/themes/snippets/commands/hotkeys
- explicit runtime mutation workflows for plugins/themes/snippets and command execution

## This skill does not cover

- note content management workflows
- Sync/Publish workflows
- devtools diagnostics and runtime JS/CDP inspection flows
- community tooling outside official CLI

## Operational limits

- desktop Obsidian app must be available
- runtime mutation commands can alter app behavior significantly
- ambiguous IDs or names must be resolved before mutation
